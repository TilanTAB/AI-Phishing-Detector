# Screenshots Guide

Add your screenshots here. The main README references these exact filenames.

## Required Screenshots

| Filename | What to Capture |
|---|---|
| `addon_home_card.png` | Gmail open → Phishing Checker sidebar → home card (before clicking Analyze) |
| `addon_results_card.png` | After clicking Analyze — results card showing Verdict, Score, Red Flags |
| `gmail_labels.png` | Gmail label list (left sidebar) showing PHISHING_DETECTED (red) and SUSPICIOUS (orange) labels |
| `script_properties.png` | Apps Script Editor → Project Settings (gear icon) → Script Properties section showing your property keys (blur/hide the values) |
| `apps_script_project_settings.png` | Apps Script Editor → Project Settings overview page |
| `gcp_pubsub_topic.png` | GCP Console → Pub/Sub → Topics → your gmail-notifications topic |
| `gcp_oauth_credentials.png` | GCP Console → APIs & Services → Credentials page showing your OAuth 2.0 client |

## Tips

- Blur or redact any API keys, secret values, account IDs, or email addresses in screenshots
- PNG format preferred (better quality for UI screenshots)
- Aim for 1200–1600px wide for good readability in the README
